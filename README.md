# fancy_tools

## Description
Outil Shell permettant d’ajouter des alias Git, automatiser les commits et installer l’environnement.

## Prérequis
- Linux
- Bash
- Git
- Compte GitHub

## Installation
git clone ...
cd fancy_tools
bash install.sh

## Utilisation
gss -> git status
gpl -> git pull
gpu -> git push
gap -> git add .

gco "message" -> commit avec préfixe automatique

updateFancyTools -> met à jour les outils
